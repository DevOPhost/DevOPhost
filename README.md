<summary>⚙️ <strong>Como ativar o Snake e o Pac-Man (GitHub Actions)</strong></summary>

<br>

Crie o arquivo `.github/workflows/animations.yml` no seu repositório **DevOPhost/DevOPhost** com o seguinte conteúdo:

```yaml
name: Generate Animations

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
  push:
    branches: [main]

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10

    steps:
      # Snake Animation
      - name: Generate Snake (dark + light)
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      # Pac-Man Animation
      - name: Generate Pac-Man
        uses: hugoalh/github-actions-core@v2
        continue-on-error: true

      - name: Generate Pac-Man (alternative)
        run: |
          curl -s "https://pacman.made4u.link/pacman.php?ghUser=DevOPhost&size=6&font=0" \
            -o dist/pacman-contribution-graph.svg || true

      # Push to output branch
      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

> Depois vá em **Settings → Actions → General** e marque **"Read and write permissions"** para o workflow funcionar.

<table>

<h3>Estágio System</h3>

<p>
Sistema web criado para apoiar a gestão de atividades de estágio, chamados de suporte, relatórios e acompanhamento de horas.
</p>

<p>
O projeto foi desenvolvido com foco em uma necessidade real: centralizar informações, facilitar o controle das atividades e tornar a rotina administrativa mais organizada.
</p>

<p>
<b>Principais recursos:</b>
</p>

<ul>
  <li>dashboard com indicadores;</li>
  <li>registro de atividades;</li>
  <li>controle de chamados;</li>
  <li>relatórios gerenciais;</li>
  <li>autenticação por perfil;</li>
  <li>estrutura separada entre frontend, backend, banco e documentação.</li>
</ul>
